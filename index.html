Table Elements

Element Name What It Does
<table> Table Holds everything that goes in the table. Contains all table elements.
<thead> Table Head Defines a set of rows that define the head of the colums of the table.
<tbody> Table Body Encapsulates a set of table rows and indicate that they comprise the body of the table.
<tr> Table Row Defines a row of cells in a table.
<td> Table Data Cell Defines a cell of a table that contains data.
<tfoot> Table Foot Defines a set of rows summarizing the columns of the table.
